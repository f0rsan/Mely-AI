# M2 视觉训练 — W9–12（含 1 周 buffer）

> 目标: 数据集管理（双入口）, AI-Toolkit LoRA 训练, 进度可视化
>
> **前置**: M1 LLM 训练完成。角色已有文字 DNA，可选择性为其绑定视觉 LoRA。

## 前置条件
- M0 + M1 完成
- AI-Toolkit 可用（按需下载框架提供）
- RTX 3070 8GB 开发机

## 数据集双入口

视觉训练支持两条独立入口，在同一 UI 中呈现：

- **入口 A（自有图片）**：用户上传自己的参考图 → 直接进入数据集管理
- **入口 B（文字生图）**：用文字描述角色外貌 → AI 生成候选训练图（需 M3 ComfyUI 完成后可完整使用；M2 阶段可先用 diffusers 提供基础能力）

---

## 任务拆解

### W9: PoC + 数据集管理

#### T2.1 — FLUX 模型 PoC (2d) ⚠️ 关键决策点
- RTX 3070 8GB 上测试：
  - FLUX-dev Q4 量化：训练能否跑通？VRAM 是否溢出？
  - FLUX-schnell：训练质量对比
- 输出：选型报告，锁定默认模型
- 验收: 有明确结论文档，选定模型在 8GB VRAM 内可训练

#### T2.2 — 入口 A：自有图片数据集导入 UI (2d)
- 拖拽批量导入图片（前端 dropzone）
- 图片预览网格
- 自动检测：分辨率、人脸清晰度（OpenCV face detection）
- 数据集质量评分（图片数量 + 分辨率均值 + 人脸检出率）
- 改进建议文字（如「缺少侧脸角度，建议补充」）
- 验收: 拖入 15 张图 → 质量评分 + 具体改进建议

#### T2.3 — 入口 B：文字生图训练数据 (1d)
- 用户输入角色外貌文字描述（或从角色 DNA 自动填充）
- 调用生图接口（M3 完成前用 diffusers 基础接口）生成 4–8 张候选图
- 用户勾选满意的图 → 加入训练数据集
- 与入口 A 图片合流 → 进入统一训练流程
- 验收: 输入描述 → 生成候选图 → 勾选 → 出现在数据集列表中

#### T2.4 — WD14 自动打标集成 (1d)
- 集成 WD14 Tagger（按需下载模型）
- 对导入图片自动打标
- 标签作为视觉 DNA 填写时的「建议值」
- 验收: 导入参考图 → 自动生成标签建议 → 显示在 DNA 表单中

### W10: 训练管线

#### T2.5 — 角色视觉 DNA 配置 UI (1d)
- 5 个下拉：发色/瞳色/肤色/风格/体型
- WD14 建议值自动填充（用户可修正）
- 若角色已有 LLM 人设，从人设文字中提取视觉描述自动填充
- 自动组合生成 `auto_prompt`
- 保存到 `character_dna` 表
- 验收: 填写 DNA → 预览自动生成的 Prompt → 保存

#### T2.6 — AI-Toolkit 训练后端 (3d)
- FastAPI 封装 AI-Toolkit
- REST API: `POST /api/visual-training/start`（接收角色 ID + 训练模式）
- 训练配置生成（轻量/标准/精细三档，同 M1 预设逻辑）
- 实时进度 WebSocket 推送
- 每 20% 进度生成一张采样预览图
- GPU 互斥锁（训练期间禁止 LLM 训练、图像生成等其他 GPU 任务）
- 训练完成后自动更新 `visual_assets` 表
- 错误处理：VRAM 溢出 → 「显存不足，请尝试轻量模式」
- 验收: 上传 15 张图 → 标准模式 → 训练完成 → LoRA 文件生成

#### T2.7 — 训练进度 UI (1d)
- 进度条 + 百分比 + 预估剩余时间
- 采样预览图实时展示（每 20% 一张）
- 训练完成 → 4 张验证图（正面/侧面/背面/特写）
- 「满意/不满意」评分按钮
- 验收: 训练全程能看到进度 + 采样图

### W11: 验证流程 + 重训

#### T2.8 — 训练验证 & 重训流程 (1d)
- 训练完成 → 自动生成 4 张标准验证图
- 不满意 → 推荐「增加训练步数」或「补充参考图」
- 一键重训按钮（保留数据集，调整参数）
- 验收: 完整的 训练 → 验证 → 重训 循环

### W12: Buffer + 集成测试

#### T2.9 — 端到端集成测试 (2–3d)
- 完整流程 A：创建角色 → 入口 A 上传图片 → DNA 配置 → 训练 → 验证
- 完整流程 B：创建角色 → 入口 B 文字生图 → 勾选 → 训练 → 验证
- 混合流程：入口 A + 入口 B 图片合并训练
- 边界情况：训练中断、VRAM 不足、图片数不够
- LLM + 视觉联动：LLM 人设 DNA 自动填充视觉 DNA 验证
- 验收: 两条入口全部跑通，LLM 人设联动正常
