# Mely AI — 决策记录

## ✅ 已确认决策

### D-001: FLUX 模型选型
- **状态**: 待 M1 W1 PoC 验证
- **计划**: RTX 3070 8GB 上对比 FLUX-dev Q4 vs FLUX-schnell（训练 + 推理质量）
- **输出**: PoC 后锁定默认模型

### D-002: 文字创角入口
- **状态**: 已确认
- **方案**: M1 同步实现。ComfyUI 文生图 → 生成 4–8 张候选图 → 用户选满意的加入训练集
- **理由**: 降低用户准入门槛——没有参考图也能开始

### D-003: DNA 自动提取
- **状态**: 已确认
- **方案**: v1 主路径为引导式手动填写（5 个下拉：发色/瞳色/肤色/风格/体型），WD14 打标作为辅助建议
- **理由**: 自动提取准确率不足以作为默认值，手动填写更可靠且教育用户理解 DNA 概念

### D-004: ComfyUI 稳定性策略
- **状态**: 已确认
- **方案**: 训练与生成互斥锁，ComfyUI 崩溃自动重启，失败给用户明确文字提示
- **Fallback**: diffusers 直接推理

### D-005: 安装体验
- **状态**: 已确认
- **方案**: 首次启动按需下载（最小视觉包 ~5GB，完整包 ~9GB），App 本体 <100MB
- **依赖**: M0 必须建好按需下载框架

### D-006: 协议安全
- **状态**: 已确认
- **方案**: Unsloth 只用 Apache 2.0 训练库（v1.5），不集成 AGPL-3.0 Studio UI
- **理由**: 规避开源协议风险
