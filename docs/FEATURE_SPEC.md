# Mely AI — 功能规格清单

| 模块 | 功能 | 验收标准 |

| LLM 环境 | Ollama 本地安装检测 & 引导 | 应用内检测 → 未安装给出引导，不暴露终端 |

| 生成历史 | 自动归档入角色库 | 每次生成自动存储 + 参数快照 |

## v1 MVP 交付标准（两条路径）

**路径 A（文字优先）**：用户能在不打开终端、不了解任何 AI 技术的情况下：写角色人设 + 上传对话样本 → 本地 LLM 微调 → 与私有角色模型对话，感受到明显的人设差异。整个流程在一个桌面窗口内完成，数据完全本地。

**路径 B（视觉优先）**：用户能在不打开终端的情况下：上传参考图（或文字生图）→ 训练角色 LoRA → 生成 5 张一致性封面图 → 导出设定书 PDF。

两条路径均可独立完成，也可组合使用（LLM 人设 → 自动填充视觉 Prompt）。

## LLM 训练参数规格

| 参数 | 轻量模式 | 标准模式 | 精细模式 |
|------|---------|---------|---------|
| 推荐数据量 | 50–200 条对话 | 200–800 条 | 800+ 条 |
| 训练步数 | 200–400 步 | 400–800 步 | 800–1500 步 |
| RTX 3070 耗时 | ~15 min | ~35 min | ~70 min |
| VRAM 需求 | ~6GB ✅ | ~6.5GB ✅ | ~7GB ✅ |
| 导出文件大小 | ~4.5GB（GGUF Q4） | ~4.5GB | ~4.5GB |

## v2 规划功能（不在 MVP 范围）

| 功能 | 说明 |
|------|------|
| 动作 LoRA 绑定 | WAN/AnimateDiff 视频角色一致性 |

| 实时语音转换 (RVC) | 直播中实时变声 |
| 角色资产导出包 | LoRA + 声纹 + 设定书打包 |

## LoRA 训练参数规格

| 参数 | 轻量模式 | 标准模式 | 精细模式 |
|------|---------|---------|---------|
| 推荐图片数 | 10–15 | 15–25 | 25–40 |
| 训练步数 | 800–1000 | 1500–2000 | 2500–3000 |
| Network Rank | 8 | 16 | 32 |
| RTX 3070 耗时 | ~15 min | ~30 min | ~55 min |
| LoRA 文件大小 | ~40MB | ~80MB | ~150MB |
| 一致性评分 | ~75% | ~88% | ~94% |
| VRAM 需求 | 8GB ✅ | 8GB ✅ | 12GB ⚠️ |

## TTS 引擎选择策略

| 阶段 | 引擎 | 优势 | 所需样本 |
|------|------|------|---------|
| v1 零样本 | F5-TTS | 零样本克隆，中英混合 | 3–10 秒 |
| v1 零样本 | CosyVoice3 | 9 语言 + 流式 150ms 延迟 | 3–15 秒 |
| v1 零样本 | IndexTTS-2 | 精确时长控制，情感分离 | 5–30 秒 |
| v1.5 精训 | Unsloth TTS | 专属模型，音色 ~92% | 20–30 min 训练 |
