# Mely AI — 功能规格清单

## v1 MVP 功能（必须交付）

### P0 — 必须有

| 模块 | 功能 | 验收标准 |
|------|------|---------|
| 角色管理 | 创建/编辑/删除角色 | 角色卡片含名称、封面图、DNA 摘要 |
| 角色管理 | 角色 DNA 参数配置 | 发色/瞳色/风格自动提取 + 手动修正 |
| 视觉训练 | 数据集导入 & 质量评分 | 检测分辨率、角度多样性、人脸清晰度 |
| 视觉训练 | 一键 LoRA 训练（轻量/标准） | AI-Toolkit，进度可视化，支持中断续训 |
| 视觉训练 | 训练采样预览图 | 每 20% 进度生成一张预览 |
| 图像生成 | 角色场景生成（单张） | 只需输入场景描述，自动注入 DNA + LoRA |
| 图像生成 | 批量生成队列 | N 场景关键词 → 后台排队，完成通知 |
| 声音绑定 | 声音参考上传 & 预览 | 上传音频 → 试听克隆效果 → 绑定确认 |
| 声音绑定 | 角色 TTS 生成（零样本） | 输入文字 → 用绑定声音输出音频 |
| 生成历史 | 自动归档入角色库 | 每次生成自动存储 + 参数快照 |

### P1 — 应该有

| 模块 | 功能 | 验收标准 |
|------|------|---------|
| 造型管理 | 造型版本创建（文字描述） | 基于自然语言差异描述新建造型分支 |
| 造型管理 | 版本树可视化 | 树状展示各造型及父子关系 |
| 生成历史 | 用途标签 & 搜索 | 按标签/日期/造型筛选历史图库 |
| 导出 | 角色设定书 PDF 导出 | 含多角度参考图 + DNA 参数 + 造型列表 |
| 安全 | LoRA 本地加密存储 | AES-256 加密，绑定设备指纹 |

### P2 — 可以有

| 模块 | 功能 | 验收标准 |
|------|------|---------|
| 安全 | 创作时间戳 & 哈希 | 训练/生成时间 + 数据哈希，可导出证明文件 |

## v1 MVP 交付标准（一句话）

用户能在不打开终端、不了解任何 AI 技术的情况下：上传照片 → 训练角色 LoRA → 生成 5 张一致性封面图 → 绑定声音 → 用角色声音朗读一段文字。整个流程在一个桌面窗口内完成，数据完全本地。

## v1.5 功能（v1 验证后 +2–3 个月）

| 功能 | 说明 |
|------|------|
| 专属声音模型精训 | Unsloth TTS 微调，音色相似度 ~92% |
| 角色对话 LLM 微调 | 基于角色设定书微调本地 Qwen/Llama |

## v2 规划功能（不在 MVP 范围）

| 功能 | 说明 |
|------|------|
| 动作 LoRA 绑定 | WAN/AnimateDiff 视频角色一致性 |
| 角色间协作生成 | 多角色同框，各自 LoRA 权重调度 |
| 实时语音转换 (RVC) | 直播中实时变声 |
| 角色资产导出包 | LoRA + 声纹 + 设定书打包 |

## LoRA 训练参数规格

| 参数 | 轻量模式 | 标准模式 | 精细模式 |
|------|---------|---------|---------|
| 推荐图片数 | 10–15 | 15–25 | 25–40 |
| 训练步数 | 800–1000 | 1500–2000 | 2500–3000 |
| Network Rank | 8 | 16 | 32 |
| RTX 3070 耗时 | ~15 min | ~30 min | ~55 min |
| LoRA 文件大小 | ~40MB | ~80MB | ~150MB |
| 一致性评分 | ~75% | ~88% | ~94% |
| VRAM 需求 | 8GB ✅ | 8GB ✅ | 12GB ⚠️ |

## TTS 引擎选择策略

| 阶段 | 引擎 | 优势 | 所需样本 |
|------|------|------|---------|
| v1 零样本 | F5-TTS | 零样本克隆，中英混合 | 3–10 秒 |
| v1 零样本 | CosyVoice3 | 9 语言 + 流式 150ms 延迟 | 3–15 秒 |
| v1 零样本 | IndexTTS-2 | 精确时长控制，情感分离 | 5–30 秒 |
| v1.5 精训 | Unsloth TTS | 专属模型，音色 ~92% | 20–30 min 训练 |
